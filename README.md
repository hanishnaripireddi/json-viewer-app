# JSON Visualizer App

A Flutter app for visualizing and navigating JSON data in a tree-like structure.

## Description

The JSON Visualizer app is designed to help users view and navigate JSON data in a tree-like structure, making it easier to understand the hierarchical relationships within the data.

## Features

- Input JSON data.
- Visualize JSON data in a tree structure.
- Expand/collapse nested JSON objects.
- Easily navigate and explore JSON data.
- JSON syntax validation.
